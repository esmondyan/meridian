## 三、各竞品详细分析

### 3.1 OSRS Wiki Prices（最强数据源）

**评分：⭐⭐⭐⭐**

**优势：**
- 数据最权威（Jagex 官方数据提供方）
- 完全免费、开源 API
- **每日交易量**数据精准（我们的 /5m API 是 5 分钟切片）
- 自动刷新（60 秒），几乎实时
- 简洁、无广告

**劣势：**
- 纯数据展示，**完全没有分析功能**
- 没有倒卖建议、没有评分
- 没有推送通知
- 没有税后利润计算
- 英文界面

**与我们的关系：**
- **我们是数据消费者**，直接调用他们的 API
- 他们提供"原材料"，我们提供"加工品"（评分+策略+推送）

---

### 3.2 GE Tracker（最大独立竞品）

**评分：⭐⭐⭐½**

**优势：**
- 用户量最大，社区成熟
- Pro 版有价格预警（$3/月）
- 7 天/30 天/90 天历史趋势
- 数据维度齐全

**劣势：**
- 有广告（免费版）
- 只有毛利，没有税后利润
- **没有倒卖评分/策略推荐**
- 英文界面
- 功能停留在"展示数据"，没有"建议动作"

**定价：**
- 免费版：基础功能 + 广告
- Pro 版：$3/月（去广告 + 预警）
- 商业版：定制价格

**与我们的差距：**

| 对比项 | GE Tracker | 本工具 |
|---|---|---|
| 用户规模 | ✅ 大 | ❌ 0 |
| 品牌信任 | ✅ 多年运营 | ❌ 新建 |
| 数据分析深度 | ❌ 展示数据 | ✅ 策略推荐 |
| 中文支持 | ❌ | ✅ |
| 推送方式 | ❌ 仅邮件 | ✅ 微信 |
| 价格 | $3/月(Pro) | 未定 |

---

### 3.3 RuneLite GE 插件

**评分：⭐⭐⭐**

**优势：**
- 嵌入游戏客户端，最便捷
- 实时数据 + 趋势图
- 免费

**劣势：**
- 仅在游戏内可用
- 功能有限（仅基础价格展示）
- 不是独立工具
- 无法做深入分析

---

### 3.4 OSBuddy

**评分：⭐（已关停）**

**说明：**
- 曾是最大的第三方客户端
- 2023 年因 Jagex 政策变动关停
- 提醒：**第三方客户端风险**（Jagex 收紧政策）

## 四、核心差异点总结

### 4.1 我们赢在哪

| 维度 | 差距 | 说明 |
|---|---|---|
| **分析深度** | 🏆 领先 | 我们是**唯一做评分+策略+税率计算**的工具 |
| **中文支持** | 🏆 绝对领先 | 所有竞品都是英文 |
| **推送能力** | 🏆 领先 | 微信推送，竞品只有邮件/网页 |
| **零门槛** | 🏆 领先 | 评级系统让不懂游戏的人也能用 |
| **定价空间** | ➡️ 优势 | 竞品最高 $3/月，我们可以做到同等或更低 |

### 4.2 我们缺什么

| 短板 | 影响 | 优先级 |
|---|---|---|
| **品牌/信任** | 新工具，玩家不敢用 | 🟡 中 |
| **用户基数** | 0 vs 竞品十万级 | 🟢 可解决（推广） |
| **实时性** | 每小时 vs 60秒 | 🟢 影响不大（倒卖不差这几分钟） |
| **收藏夹** | 有需求 | 🟢 可加 |
| **价格预警** | GE Tracker 有 | 🟡 重要功能 |

### 4.3 我们的终极卖点

```
竞品：  "这是数据，你自己分析"
我们：  "这是建议，你直接执行"
```

这个差异是本质性的：
- GE Tracker 给一堆表格，你得自己看
- 我们给**A+ / A / B / C / D 评级**，一眼知道该倒什么
- 我们还给**税后利润**（竞品都没算 GE 2% 税）
- 我们还给**本金输入框**，算你买得起什么
